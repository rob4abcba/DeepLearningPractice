## AND, OR Model

**Structure**:

![AND, OR Model](https://pic.pimg.tw/darren1231/1483983081-1325337865.png)

Step Function

$$
Y = \left\{ \begin{matrix} 1,\text{if }\sum_{i}^{}{a_{i}W_{i} \geq \theta} \\ 0,\text{if }\sum_{i}^{}{a_{i}W_{i} < \theta} \\ \end{matrix} \right.
$$
